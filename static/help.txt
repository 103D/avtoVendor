🔐 СПРАВКА ПО ПОЛУЧЕНИЮ JWT ТОКЕНА
=====================================

ПРОБЛЕМА:
--------
Учётные данные "Lassio / Lassio@123" не действительны на API.
Система ожидает действительных креденшалов вашего аккаунта.

РЕШЕНИЕ 1: ВСТАВИТЬ СУЩЕСТВУЮЩИЙ ТОКЕН
--------------------------------------
1. Откройте страницу логина системы
2. Найдите поле "Вставить токен вручную"
3. Скопируйте ваш JWT токен
4. Вставьте его в поле
5. Нажмите "Использовать этот токен"

РЕШЕНИЕ 2: ПОЛУЧИТЬ НОВЫЙ ТОКЕН
------------------------------
1. Введите ваши ДЕЙСТВИТЕЛЬНЫЕ креденшалы Safia Delivery
2. Нажмите "Получить токен через API"
3. Если успешно, токен будет отображён

РЕШЕНИЕ 3: ИЗВЛЕЧЬ ИЗ БРАУЗЕРА
-----------------------------
1. Откройте Safia Delivery в браузере
2. Залогиньтесь со своими креденшалами
3. Откройте DevTools (F12)
4. Перейдите в Application → Cookies
5. Найдите и скопируйте JWT токен
6. Используйте его в нашей системе

КАК УЗНАТЬ СВОИ КРЕДЕНШАЛЫ?
--------------------------
Свяжитесь с администратором Safia Delivery или:
- Посетите: https://orderconfirmer.safiadelivery.com/
- Напишите: support@safiadelivery.com

ПРОВЕРКА ТОКЕНА
---------------
Используйте Python скрипт:

    python vendor/get_jwt_token.py <ваше_имя> <ваш_пароль>

ИЛИ проверьте через curl:

    curl -X GET \
      "https://orderconfirmer-api.safiadelivery.com/api/menu/Menus" \
      -H "Authorization: Bearer ВАШ_ТОКЕН"

Если статус 200 - токен действителен!
Если статус 401 - токен истёк или невалиден.

ВАЖНО
----
- Токены имеют ограниченный срок действия (обычно 1 час)
- Не делитесь вашими токенами и креденшалами с другими
- Каждый раз при необходимости получайте новый токен
