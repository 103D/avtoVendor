# Веб-интерфейс ЛАССИО - Этап 1

Веб-приложение для управления товарами: загрузка Excel файла, трансформация данных в таблицу, заполнение фактических количеств.

## Функциональность

**Шаг 1:** Загрузить Excel файл (ЛАССИО)
- Drag & Drop или выбрать файл
- Валидация: .xlsx/.xls/.xlsm, название содержит "лассио"

**Шаг 2:** Обработать файл
- Парсинг Excel через pandas
- Выбор столбцов: D (SKU), G (Название), K (Кол-во по накладной)
- Преобразование в JSON

**Шаг 3:** Заполнить таблицу
- 5 столбцов: № | SKU | Название | По накладной | **По факту** (редактируемое)
- Валидация в реальном времени (зеленый/красный border)
- Ввод: числа или "нету"

**Шаг 4:** Сохранить
- Финальная валидация на клиенте и сервере
- Данные сохраняются в session для этапа 2

## API

| Метод | Endpoint | Параметры | Описание |
|-------|----------|-----------|---------|
| GET | / | - | Главная страница (stage1.html) |
| POST | /api/upload-file | file (multipart) | Загрузка Excel файла |
| POST | /api/transform-file | - | Обработка файла в JSON |
| POST | /api/save-stage1-data | JSON массив | Сохранение заполненных данных |
| GET | /api/get-session-data | - | Получение данных из session |

## Проблемы

| Проблема | Решение |
|----------|---------|
| Файл не загружается | Проверить расширение (.xlsx/.xls/.xlsm), в названии "лассио", размер < 50MB |
| Таблица не появляется | Проверить консоль браузера (F12) и PowerShell на ошибки |
| Поля не редактируются | Перезагрузить страницу (F5), очистить кэш |
| Приложение не запускается | Убедиться: pip install выполнен, Python 3.7+, порт 5000 свободен |
